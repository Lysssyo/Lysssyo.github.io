**Phase 1 · JS + TS（2-3 周）** 以 zh.javascript.info 为主线，重点攻克闭包、原型链、async/await、事件循环这些和你已有语言经验差异大的部分。基础语法快速扫过即可。JS 核心概念到位后，紧接着学 TypeScript，官方文档就够，有类型系统概念上手很快。

**Phase 2 · React（2-3 周）** 跟 react.dev 官方文档学，全部基于函数组件和 Hooks。掌握 useState、useEffect、useRef 这几个核心 Hook，理解组件化思维和 JSX。同时用 Tailwind CSS 写样式，在实践中自然学会 Flexbox/Grid 布局，不用单独开一个阶段学 HTML/CSS。

**Phase 3 · Next.js 全栈（3-4 周）** 先跟 nextjs.org/learn 官方教程做一个完整的 dashboard 应用。然后学习 Prisma（ORM）、NextAuth（认证）、API Routes 等全栈必备工具。同时带着具体问题去读 Taxonomy 和 Bulletproof React 的源码作为参考。

**Phase 4 · 实战项目（4+ 周）** 用 Next.js + TypeScript + Tailwind + Prisma 做一个完整的全栈应用，包含用户认证、数据库 CRUD、前后端交互、部署上线。如果想结合 AI 方向，可以参考 fullstack-ai-nextjs 做一个 AI 驱动的应用。
