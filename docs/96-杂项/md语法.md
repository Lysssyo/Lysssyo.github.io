1. **HTTP 状态码**

    （这里空一行，下面开始缩进 4 格写文本）
    状态码由三位数字组成，第一个数字定义了响应的类别。

    - 200 OK：请求成功
    - 404 Not Found：找不到资源
    - 500 Internal Server Error：服务器内部错误

2. **HTTP 请求方法**

    常用的请求方法如下：
    
    - GET：获取资源
    - POST：提交数据
